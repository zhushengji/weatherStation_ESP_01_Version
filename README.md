# weatherStation_ESP_01_Version
基于esp01与ssd1306驱动的0.96寸128*64分辨率的OLED实现的桌面气象站
硬件地址：https://oshwhub.com/zhushengji/weatherstation
本项目灵感来自B站@发明控，本人重构了全部代码用作教学。

项目功能如下：

1.自动配网

2.强制门户认证

3.天气预报获取

4.时间日期获取

5.OTA（代码完整但本项目未启用）

本项目完全开源，就是一个新手教学项目，希望大家好好学习，天天向上！比心~
